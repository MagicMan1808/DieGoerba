# Assignment 10 – Course Feedback System (DieGoerba)

## Projektübersicht
Dieses Projekt ist Teil des Moduls **GMCI** und stellt einen webbasierten Prototyp
für ein **anonymes, sessionspezifisches Course-Feedback-System** dar.
Studierende können während oder nach einer Veranstaltung schnell Feedback geben,
während Lehrende Live-Indikatoren und übersichtliche Auswertungen erhalten.

Der Fokus von **Assignment 10** liegt auf der **Umsetzung eines klickbaren Web‑Prototyps**
auf Basis von HTML, CSS und JavaScript.

---

## Ziel von Assignment 10
- Umsetzung der Kernfunktionen als Web‑Prototyp
- Anwendung von HTML5, CSS und JavaScript
- Interaktive Oberfläche für Studierende und Lehrende
- Keine vollständige Backend‑Implementierung (Simulation erlaubt)

---

## Funktionen des Prototyps

### Studierende
- Zugriff auf eine Session über QR‑Code oder Link
- Abgabe von anonymem Feedback
  - Skalenfragen
  - Emoji‑Feedback
  - optionales Freitextfeld
- Anzeige einer zusammengefassten Gruppenübersicht nach der Session

### Lehrende
- Erstellen und Starten von Sessions
- Anzeige eines Live‑Feedback‑Dashboards
- Einsicht in Wochen‑ bzw. Sessionsübersichten
- Einfache Auswertung der Rückmeldungen
